

## Installation

- Install Node.js and NPM

- Installation dependencies

  ```bash
  npm install
  // or use yarn
  yarn
  ```

- Start

  ```bash
  npm start
  // or use yarn
  yarn start
  ```

## Usage

For basic usage 

```bash
node index.js
```

For with 100 subs.

```bash
node index-hundred-subs.js
```
